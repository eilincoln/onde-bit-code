Explorando as Tags <br> e <hr>

O que são as tags <br> e <hr>?

As tags <br> e <hr> são elementos HTML usados para manipular a estrutura de uma página web. Elas são frequentemente usadas para criar quebras de linha e linhas horizontais, respectivamente.

A Tag <br>

A tag <br> é usada para inserir uma quebra de linha em um texto ou em um bloco de conteúdo HTML. Ela não possui uma tag de fechamento, pois é uma tag vazia, ou seja, não possui conteúdo próprio.

Exemplo de uso:

<p>Este é um parágrafo.<br>Esta é uma nova linha.</p>

Isso resultará em:

Este é um parágrafo.
Esta é uma nova linha.

A Tag <hr>

A tag <hr> é usada para criar uma linha horizontal para separar conteúdo em uma página web. Ela também é uma tag vazia, não requer um elemento de fechamento e é frequentemente usada para dividir seções de uma página.
