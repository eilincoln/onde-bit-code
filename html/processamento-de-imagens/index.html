Processamento de imagem

Utilizar os formatos corretos

JPEG: formato de mais qualidade, porém mais pesado

PNG: formato um pouco inferior, porém, mais comprimido. Porém é boa para transparência

WEBP: formato criado pelo google, para ser melhor usado nas páginas web.

SVG: É um vetor que são imagem geométrica super leve!
